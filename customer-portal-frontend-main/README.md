# 🌟 Customer Portal 

A Customer Portal application that provides users with seamless login, personalized profile management, loan details, document uploads, and customer support – all built on **Next.js** and microfrontend architecture.

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## 🎯 Features  

- ✅ **Mobile Number Login**: Quick access using OTP-based login.  
- ✅ **Personalized Dashboard**: Displays user details like Customer ID, Age, and more.  
- ✅ **Loan Information**: View and manage loan-related details.  
- ✅ **Document Management**: Upload and manage important documents.  
- ✅ **Customer Support**: Direct communication with support representatives.
